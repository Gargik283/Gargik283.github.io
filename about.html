About Me

I have used java as the programming language for writing code in ARDUNIO IDE software. The components used for making this projects are arduino,buzzer,ultrasonic sensor,breadboard,jumpers and wires. I'm proficient in using tools such as EDA,FPGA,VERILOG,Xilinx.

I'm a quick learner and enthusiastic about contributing to innovative VLSI projects. I'm actively seeking opportunities to apply my knowledge and develop my skills.
